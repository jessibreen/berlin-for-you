# Berlin for You ♥

This is a location-aware Berlin map for serendipitous travel.

It is designed to be simple, readable on a phone, and useful while walking around the city. The POIs include a mix of tourist highlights, cafes, and odd urbanities.

Gute Reise Andy!

## What this app does

- Loads POIs from `data/pois.geojson`
- Centers the map on the POI extent automatically
- Shows a popup with each place's name and notes
- Supports live location tracking
- Triggers proximity alerts (toast, vibration, browser notification when allowed)
- Uses a high-contrast, gritty Berlin-inspired UI theme
